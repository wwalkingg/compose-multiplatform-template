# Compose Multiplatform Template

### Modules

#### android

这是 Android 应用程序模块。

#### desktop

这是桌面应用程序模块，运行在 JVM 中，包括平台 Windows、Linux 和 macOS。

#### core-ui

该模块负责显示 UI 界面，包括响应式变化和路由。

#### core-util

该模块包含通用的工具类或方法。

#### core-model

该模块包含应用程序所使用的数据模型。

#### core-network

该模块包含通用的网络工具。

#### feature

该模块包含应用程序的主要功能或组件，例如支付功能和购物车功能等。

#### libs

这里包括一些其他库。
