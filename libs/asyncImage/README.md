# AsyncImage

a lib to load all source image.

## Supported Data Types

- String (mapped to a Uri) ⚒️
- HttpUrl ⚒️
- Uri (android.resource, content, file, http, and https schemes) ⚒️
- File ⚒️
- @DrawableRes Int ⚒️
- Drawable ⚒️
- Bitmap ⚒️
- ByteArray ⚒️
- ByteBuffer ⚒️

## Supported Image Formats

- BMP ⚒️
- JPEG ⚒️
- PNG ⚒️
- WebP ⚒️
- HEIF (Android 8.0+) ⚒️
- SVG ⚒️
- GIF ⚒️
